# Pull Request: Добавление подвала (footer)

## Что сделано
- В файл `index.html` добавлен тег `<footer>` с информацией об авторских правах.
- В `style.css` добавлены стили для оформления подвала (темный фон, белый текст, фиксированное положение внизу страницы).
- Обновлены отступы основного контента, чтобы подвал не перекрывал текст.

## Как это проверить
1. Переключиться на ветку `feature/add-footer`.
2. Открыть `index.html` в браузере.
3. Убедиться, что внизу страницы отображается темная полоса с текстом "© 2026 Проект Propeler. Все права защищены.".
4. Проверить, что подвал остается внизу при изменении размера окна.
