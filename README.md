# Project Job Processing System

Система для обработки фоновых задач с использованием Celery и Redis.

## Основные возможности
- Выполнение задач по анализу слов (`WORD_STATS`).
- Сравнение результатов двух задач анализа (`WORD_STATS_COMPARE`).
- REST API для управления задачами.
- Мониторинг статуса выполнения.

## Стек технологий
- **Python 3.11**
- **FastAPI**: Веб-фреймворк.
- **Celery**: Очередь задач.
- **Redis**: Брокер сообщений.
- **PostgreSQL**: База данных.
- **SQLAlchemy**: ORM.
- **Alembic**: Миграции БД.
- **Docker**: Контейнеризация.

## Структура проекта
```text
propeler/
├── core/               # Конфигурация и исключения
├── models/             # SQLAlchemy модели и Enum
├── repositories/       # Логика работы с БД
├── routes/             # API эндпоинты
├── tasks/              # Celery задачи и Executor
│   └── handlers/       # Обработчики конкретных типов задач
├── alembic/            # Миграции базы данных
├── main.py             # Точка входа в приложение
├── requirements.txt    # Зависимости
├── Dockerfile          # Сборка образа
└── .env                # Переменные окружения
```

## Переменные окружения проекта
Все настройки хранятся в файле `.env`. Пример заполнения в `.env.example`.

## Как запустить проект
1. Склонируйте репозиторий.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите Docker-контейнеры:
   ```bash
   docker-compose up --build
   ```
4. Приложение будет доступно по адресу `http://localhost:8000`.
